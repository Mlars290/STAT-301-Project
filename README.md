# 📊 STAT 301 ANOVA Project

## 📌 Overview
This project applies ANOVA (Analysis of Variance) methods to analyze relationships between variables within a dataset and draw data-driven conclusions.

## 🎯 Objectives
- Formulate research questions using real-world data  
- Apply ANOVA statistical testing  
- Interpret p-values and test statistics  
- Communicate findings clearly and effectively  

## 📂 Project Structure
This repository will include:
- Final project report  
- Dataset used for analysis  
- Code and/or statistical outputs  
- Visualizations (graphs and tables)  

## 🧪 Methods
The analysis uses ANOVA to compare group means and determine whether statistically significant differences exist between variables.

## 📈 Results
Results will include:
- Test statistics  
- p-values  
- Hypothesis decisions based on significance level (α = 0.05)  

## ⚠️ Notes
This project is part of STAT 301 and will be updated as the analysis is completed.
