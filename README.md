# Sleep Duration Analysis Using Two-Way ANOVA

## 📊 Overview
Analyzed how stress level and BMI category influence sleep duration using statistical modeling techniques.

This project applies a two-way ANOVA to uncover both independent and interaction effects between variables in a real-world health dataset.

---

## 🔑 Key Insights
- Sleep duration **decreases significantly as stress increases**
- BMI category has a **statistically significant impact on sleep duration**
- A **significant interaction effect** shows that the relationship between BMI and sleep changes at higher stress levels
- At high stress levels, the expected pattern between BMI groups **reverses**, indicating a crossover interaction

---

## ⚙️ Methods
- Data cleaning and preprocessing
- Feature engineering (categorization of stress levels)
- Two-way ANOVA using a linear model (`lm`)
- Type II ANOVA for unbalanced data
- Post-hoc analysis using Welch t-tests
